<objective>
Land `sync_profile(profile_id, app_state)` — the orchestration routine that takes a profile_id, decrypts its PAT, instantiates a DiscogsographyClient, streams the paged collection into a TEMP staging table via psycopg3 COPY, performs the atomic staging-swap in a single transaction, refreshes in-process caches inline (D-14), and surfaces typed errors back to the caller (the Plan 04 admin endpoint). src/gruvax/sync/profile_sync.py:
  async def sync_profile(profile_id: str, app_state) -> dict:
      """Stream-fetch the profile's discogsography collection, staging-swap the
      local cache, and refresh in-process caches. Returns:
        {"status": "ok", "item_count": int, "took_ms": float, "user_id": str}
      Raises:
        - SyncInProgress if pg_try_advisory_lock returns FALSE.
        - PATRejected if the PAT is rejected (also sets app_token_revoked=TRUE).
        - RateLimitExhausted / ServerError / NetworkError on upstream failures.
      """

<!-- From CONTEXT.md §specifics + RESEARCH.md §Pattern 2/3/7 — the canonical sequence -->

1. Acquire dedicated psycopg.AsyncConnection (NOT a pool slot — Pitfall 6).
2. SELECT pg_try_advisory_lock(_lock_key(profile_id)). If FALSE → raise SyncInProgress
   (with the Pitfall-1 defensive check: if last_sync_status='in_progress' AND
   last_sync_at < now() - INTERVAL '5 minutes', surface a clearer stale-lock message).
3. try:
     a. UPDATE gruvax.profiles SET last_sync_status='in_progress', last_sync_error=NULL
        WHERE id = profile_id.
     b. SELECT app_token_encrypted, discogsography_user_id, deleted_at FROM gruvax.profiles
        WHERE id = profile_id. If app_token_revoked=TRUE before the call AND ciphertext is
        the sentinel '\x' placeholder → raise PATRejected (PAT never set).
     c. pat = pat_crypto.decrypt_pat(ciphertext)  # raises InvalidToken on key mismatch
        → caller catches and translates to PATRejected + last_sync_error='pat_rejected'.
     d. client = DiscogsographyClient(base_url=settings.DISCOGSOGRAPHY_BASE_URL, pat=pat).
     e. CREATE TEMP TABLE profile_collection_staging (...) ON COMMIT DROP.
     f. envelope = await client.first_page(); user_id = str(envelope["user_id"]).
        (Pitfall: validate that releases[0]["catalog_number"] is a key in the envelope —
        if discogsography rolled back catalog# exposure, fail loudly here BEFORE writing.)
     g. async with cur.copy("COPY profile_collection_staging (release_id, folder_id, ...)
        FROM STDIN") as copy:
          for release in envelope["releases"]: await copy.write_row(_release_to_tuple(release))
          if envelope["has_more"]: stream subsequent pages via client._get_page loop.
     h. async with conn.transaction():
          DELETE FROM gruvax.profile_collection WHERE profile_id = %s::uuid;
          INSERT INTO gruvax.profile_collection (profile_id, release_id, folder_id, artist,
              title, label, catalog_number, year)
            SELECT %s::uuid, release_id, folder_id, artist, title, label, catalog_number, year
            FROM profile_collection_staging;
          UPDATE gruvax.profiles SET
            last_sync_at = NOW(),
            last_sync_status = 'ok',
            last_sync_item_count = %s,
            last_sync_error = NULL,
            discogsography_user_id = COALESCE(discogsography_user_id, %s::uuid),
            app_token_revoked = FALSE
            WHERE id = %s::uuid;
          (Transaction commit on exit of `async with conn.transaction()`.)
     i. await _refresh_app_caches(app_state) — D-14 inline.
   except PATRejected:
     UPDATE profiles SET last_sync_status='failed', last_sync_error='pat_rejected',
                         app_token_revoked=TRUE WHERE id=%s::uuid; re-raise.
   except RateLimitExhausted:  set last_sync_error='rate_limited'; re-raise.
   except ServerError:         set last_sync_error='server_error'; re-raise.
   except NetworkError:        set last_sync_error='network'; re-raise.
   except Exception:           set last_sync_status='failed', last_sync_error=NULL; re-raise.
   finally:
     SELECT pg_advisory_unlock(_lock_key(profile_id)).
     await client.aclose() if client constructed.
     await conn.close().
