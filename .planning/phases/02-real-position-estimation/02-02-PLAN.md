async def search_collection(pool, q, limit) -> tuple[list[SearchRow], float]
  - WITH fts AS (... ts_rank_cd(v.fts_vector, tsq.query, 4) AS score ... websearch_to_tsquery('english', %s) ...)
  - cat AS (... 0.9::float AS score ... WHERE lower(regexp_replace(catalog_number,'[\s\-_./]+','','g')) LIKE lower(regexp_replace(%s,...)) || '%%' ...)

| T-02-08 | Denial of Service | trigram similarity over all label/artist terms on near-miss | mitigate | did_you_mean_query runs ONLY when FTS returns empty; `q` capped at 200 chars (existing T-01-10); `LIMIT 1`; relies on discogsography GIN index where present |
| T-02-09 | Information Disclosure | did_you_mean suggestion text | accept | Suggestions are owned-collection label/artist terms only; no PII; home-LAN only |
