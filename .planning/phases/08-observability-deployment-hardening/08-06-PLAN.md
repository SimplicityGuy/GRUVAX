| Docker image -> baked secrets | SESSION_SECRET in CI is a throwaway literal, never a real secret |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-08-21 | Denial of Service | host disk via container logs | mitigate | json-file driver max-size 10m + max-file 3 caps each service at ~30 MB (DEP-04). |
| T-08-22 | Information Disclosure | CI SESSION_SECRET | accept | CI uses a non-real test literal; no production secret enters the workflow file or logs. |
